## ToggleButton

### System props

<SystemProps props={['margin']} />

### Custom props

| Name       | Type              | Default  | Description                                        |
| :--------- | :---------------- | :------: | :------------------------------------------------- |
| `variant`  | 'normal' \|'fade' | `normal` | Variant to display                                 |
| `name`     | String            |  `null`  | `name` attribute to be passed down to the children |
| `children` | ReactElement      |  `null`  | Children should be `Radio` Elements                |
